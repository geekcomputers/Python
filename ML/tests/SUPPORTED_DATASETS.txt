✓ cifar100 / cifar-100 / cifar_100
✓ mnist
✓ fashion_mnist / fashion-mnist / fashionmnist

✓ stl10 / stl-10 / stl_10
✓ tiny_imagenet / tiny-imagenet / tinyimagenet

Large Datasets:
✓ food101 / food-101 / food_101

- Type "fashion-mnist" or "fashionmnist" → Works!
